Explore the available API endpoints:

API Overview: GET /api/
Regions: GET /api/regions/
Weather Parameters: GET /api/parameters/

Weather Statistics: GET /api/weather/statistics/

Query Parameters

  • ?region=UK - Filter by region code
  • ?parameter=Tmax - Filter by parameter code
  • ?year_from=2020&year_to=2023 - Filter by year range

Example Usage
